# discord-quest-bypass
## Getting started
### Note: This might not work on ur browser for apps that require playing a game, use the offical desktop client

1. Accept **ONE** Quest of choice, I recommened a video if you are just trying this out.
2. Press `Ctrl` + `Shift` + `I` at the same time to open the console
3. Discord will warn you about pasting malicious scripts inside and will not permit pasting unless you type "allow pasting" and execute it
4. Paste the script provided above by copying (`Ctrl` + `C`) and pasting it inside the console (`Ctrl` + `V`), press enter to execute it
5. Wait.
6. Accept Rewards
7. Repeat for all quests you want

Enjoy.
